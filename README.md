# Uber Fleet Performance Analysis Dashboard using POWER BI

### Project Title
<b>Uber Fleet Performance Analysis Dashboard</b>

### Project Overview
The <b>Uber Fleet Performance Analysis Dashboard</b> is a comprehensive multi-page reporting solution designed to track and analyze the operational efficiency, reach, and utilization of various vehicle types within the Uber fleet. This project provides a granular view of performance metrics across multiple services, including Uber Auto, Uber-X, Uber Moto, Uber Taxi, and Uber GO XL.

### Purpose
The purpose of this dashboard is to evaluate fleet operations through:<br>

â€¢ <b>Demand Mapping:</b> Tracking top pickup and drop-off locations to identify geographical hotspots.<br>
â€¢ <b>Usage Categorization:</b> Distinguishing between business and personal travel purposes to understand customer intent.
